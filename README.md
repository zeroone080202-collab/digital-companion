# 디지털 동행

고령층이 음식 주문 키오스크와 스미싱 문자 판별을 안전하게 연습할 수 있는 정적 웹사이트입니다.

## 주요 기능

- 큰 글자와 단순한 화면 구성
- 글자 크기 조절
- 고대비 모드
- 한국어 음성 안내
- 음식 주문 키오스크 연습
- 스미싱 문자 판별 퀴즈
- 연습 결과 브라우저 저장
- 모바일 및 태블릿 화면 대응
- 실제 결제나 개인정보 입력 없음

## 실행 방법

별도의 설치가 필요하지 않습니다.

1. 저장소를 다운로드합니다.
2. `index.html`을 브라우저에서 엽니다.

또는 VS Code의 Live Server 확장 기능으로 실행할 수 있습니다.

## GitHub Pages 배포

1. GitHub에서 새 저장소를 만듭니다.
2. `index.html`, `style.css`, `app.js`, `README.md`를 저장소 최상위에 업로드합니다.
3. 저장소의 `Settings` → `Pages`로 이동합니다.
4. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
5. Branch를 `main`, 폴더를 `/(root)`로 선택하고 저장합니다.
6. 잠시 후 표시되는 주소로 접속합니다.

## 데이터 저장

학습 결과와 화면 설정은 브라우저의 `localStorage`에 저장됩니다. 서버나 데이터베이스는 사용하지 않습니다.

## 대회 시연 추천 순서

1. 첫 화면에서 글자 확대와 고대비 기능 시연
2. 음식 주문 연습 진행
3. 잘못된 버튼을 일부러 눌러 친절한 오류 안내 시연
4. 음성 도움 기능 시연
5. 스미싱 판별 퀴즈 진행
6. 나의 연습 기록 확인
